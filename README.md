#  Good Glory Schools Admissions Portal

##  Project Overview

The **Good Glory Schools Admissions Portal** is a front-end web application designed to simplify and modernize the student application process. It allows students to:

* Explore schools
* Fill in academic and personal details
* Save application data locally
* View their profile summary
* Navigate through a clean, interactive interface

The platform focuses on clarity, simplicity, and user experience, making the admissions journey smooth and engaging.


##  Features

### 1. Landing Page

* Clean and modern UI using **Tailwind CSS**
* Smooth scroll animations and transitions
* Call-to-action buttons guiding users to apply
* Responsive navigation bar

### 2.Application Form (Applies.html)

* Collects:

  * Personal information
  * Academic details (KCPE & KCSE)
  * Contact information
* Interactive features:

  * Profile image upload
  * Level selection (KCPE / KCSE)
  * Input validation
  * Save & discard functionality
* Data stored using **localStorage**

### 3. Profile Page (Profile.html)

* Displays saved user data dynamically
* Shows:

  * Personal details
  * Academic records
  * Uploaded profile image
* Pulls data from **localStorage**

### 4.  Schools Page (Schools.html)

* Displays recommended institutions
* Card-based UI with:

  * Images
  * Ratings
  * Locations
* Simple and visually appealing layout

### 5.  JavaScript Functionality

* DOM manipulation
* Event handling
* Form validation
* Local storage (save & retrieve data)
* UI interactions:

  * Button loading states
  * Scroll animations
  * Image preview upload


##  Technologies Used

* **HTML** – Structure
* **CSS + Tailwind CSS** – Styling & layout
* **JavaScript** – Functionality & interactivity
* **Font Awesome** – Icons
* **LocalStorage API** – Data persistence

##  How to Run the Project

1. Download or clone the project
2. Open the folder
3. Double-click:
   index.html
4. Navigate through pages using the navbar
5. Fill the form and click **Save**
6. View saved data on the **Profile page**
